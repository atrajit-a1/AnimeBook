About This Library

The Anime Book Library is a community-driven project that aims to collect and showcase amazing anime-style stories from contributors around the world.

All stories are stored as markdown files in our GitHub repository, making it easy for anyone to contribute their own stories or improve existing ones.

The website dynamically fetches and renders these stories using the GitHub API, providing a seamless reading experience for all anime enthusiasts.

How to Contribute

1

Fork the Repository

Start by forking our GitHub repository to your account.

2

Create Your Story

Write your story in markdown format with proper frontmatter.

3

Add to Right Category

Place your markdown file in the appropriate category folder.

4

Submit a Pull Request

Submit your changes to the content branch for review.
